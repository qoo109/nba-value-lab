# Gold 5,826 Freeze Manifest Output Contract v1

Allowed output is aggregate only:

```text
policy and schema identifiers
Artifact/file bindings
row and column counts
schema/table/metadata/corpus SHA-256 digests
season labels
validation booleans
scientific boundaries
formal Stake
```

Forbidden output:

```text
game IDs or dates
team or feature identifiers
raw/sample rows
row-level hashes
individual feature values
player information
market prices
```

Maximum JSON size: `1,048,576` bytes.
