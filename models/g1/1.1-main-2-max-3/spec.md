# NBA Value Lab G1.1 — 主要場次 2 場、最多 3 場

## 目的

G1.1 保留 G1 FINAL 的雙向檢查、價格分層、資料 Gate、T−60m 鎖定與 T−5m 最終複核，只調整主要場次數量政策。

## 數量政策

- 一般目標：每批 2 場主要場次。
- 硬上限：每批最多 3 場。
- 仍允許 0 或 1 場；不得為了湊數降低 Gate。
- 第一、第二場必須通過全部既有 G1 主要場次硬閘門。
- 第三場除了全部既有硬閘門外，還必須通過額外嚴格門檻。

## 第三場額外門檻

- 資料覆蓋率至少 90%。
- 勝率區間寬度不超過 5pp。
- 至少 4 個有效比較來源。
- G1 主要緩衝至少 2pp。
- 新聞風險必須為 0。
- 信心水準必須為高。

## 排序

排序仍依序考慮資料新鮮度、傷病與輪替確定性、區間寬度、主要緩衝、模型市場差解釋、反向路徑與相似案例穩定性。不得只按 EV 或中性勝率排序。

## 階段

- T−60m：鎖定 0～3 場「主要場次待 T−5m 確認」。
- T−5m：重新跑全部 Gate，鎖定 0～3 場「最終主要場次」。
- Closing：只用於驗證，不回寫賽前預測。

## 研究限制

正式投注額維持 0。G1.1 仍屬模型驗證與紙上測試。
