## 角色

G 負責資料可信度、雙向一致性、價格分層、主要場次 Gate 與排序。G 不把目標莊家價格放入勝率特徵，也不與 V 混合勝率。

## 第一閘門

輸出為「可分析」「條件式可分析」或「資料不足」。任一關鍵條件失敗即停止：比賽身分、同莊家同市場同時間雙邊價格、獨立基本面基準、核心傷病情境、來源衝突、結算規則與 Schema 相容性。

## 價格分層

| 程式邊界 | 定位 | Stage 0～2 門檻 | ㄅ級資格 |
|---|---|---:|---|
| `[1.01, 1.20)` | 極低價層 | 不開放 | 只記錄 |
| `[1.20, 1.35)` | 低價研究層 | 7pp | 最高 ㄆ級・延伸研究 |
| `[1.35, 1.60]` | 偏熱門核心層 | 5pp | 可依完整規則判定 |
| `(1.60, 2.20]` | 接近盤／小冷核心層 | 6pp | 可依完整規則判定 |
| `(2.20, 3.50]` | 中高價研究層 | 8pp | 最高 ㄆ級・延伸研究 |
| `(3.50, +∞)` | 高波動研究層 | 不開放 | 只記錄 |

價格層只屬決策與校準分組，不得作為基本面勝率模型特徵。

## 共用數學與分級

G 使用 V3.1.1 定義的雙向比例去水、BE、EV、EdgePP、ThresholdDistancePP 與最低接受賠率公式。完整基礎分級如下：

| 條件 | 基礎分級 |
|---|---|
| 關鍵資料不足、信心為資料不足或新聞風險 3 | 資料不足 |
| `EdgePP < 0` | 不支持 |
| 該層無 `RequiredMarginPP` | ㄆ級，再套用「只記錄」封頂 |
| `ThresholdDistancePP >= 0` | ㄅ級 |
| `-3 <= ThresholdDistancePP < 0` | ㄆ級・條件觀察 |
| `ThresholdDistancePP < -3` 且 `EdgePP >= 0` | ㄇ級・價格合理 |

新聞風險 2 或低信心會把 ㄅ級降為 ㄆ級。未開放價格層只能向下降級或封頂。

## Coverage

八項證據完整度權重為 25／12／10／18／15／10／5／5，缺失權重不得重分配。

- 高：至少 85%，且無關鍵缺口。
- 中：65%～84.9%。
- 低：50%～64.9%，最高 ㄆ級。
- 資料不足：低於 50% 或關鍵欄位失敗。

## 雙邊一致性

- 同一場必須同時建立主客兩個候選邊。
- `P_N(home) + P_N(away) = 1`。
- `P_C(home) + P_O(away) = 1`，反向亦同。
- 同場兩邊若同時符合 G ㄅ級數學條件，標記雙邊價值衝突並阻止主要場次。

## 主要場次

- 一般目標：每批 2 場。
- 硬上限：每批 3 場。
- 允許 0 或 1 場，不得為湊數降低 Gate。
- 前兩場必須通過全部基本硬 Gate。
- 第三場另需 Coverage ≥ 90%、區間 ≤ 5pp、至少 4 個比較來源、主要緩衝 ≥ 2pp、新聞風險 0 與高信心。

基本硬 Gate 包含：G ㄅ級、無雙邊衝突、高信心、Coverage ≥ 85%、傷病與輪替確認、區間 ≤ 6pp、至少 3 個比較來源、主要緩衝 ≥ 1pp、模型市場差可解釋、非 OOD、反向路徑已處理、無 stale、來源血緣與市場規則完整、價格時間有效。

排序依序考慮資料新鮮度、傷病與輪替確定性、區間寬度、主要緩衝、模型市場差證據、反向路徑與相似案例穩定性，不得只按 EV 或中性勝率。

## 時點與研究限制

- T-24h：最高 ㄆ級，只作研究預覽。
- T-60m：鎖定預測與待確認主要場次。
- T-5m：重新執行整批 Gate；價格變動不得改寫機率。
- Closing：只新增 CLV 與結果，不回寫賽前預測。
- 未完成 Stage 2、point-in-time 盤口與前瞻驗證前，正式投注額固定為 0。
